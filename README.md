# Tensor-Link Utility (TLU)

> **"Projecting Domain Complexity into Mathematical Clarity."**

TLU is a high-fidelity mathematical analysis pipeline designed to project transaction-based domain data (such as financial ledgers, communication logs, or supply chain flows) into an $N \times N$ tensor space. By applying models from **Kinematics (Robotics), Thermodynamics, Information Geometry, Control Theory and so forth**, TLU uncovers hidden structural dynamics that traditional statistics often miss.

-----

## System Architecture

TLU follows the **Unix Philosophy**: small, specialized filters connected via streams.

  * **Phase 1: Projection**: Transforming raw CSV/JSON into integer-indexed COO/Tensor formats.
  * **Phase 2: Core Analysis**: Pure mathematical filters (No side effects, NumPy-driven).
  * **Phase 3: Presentation**: High-density visualization using the "Fail-Fast" Theme Engine.

## Quick Start

TLU is fully containerized. You can go from zero to a full 3D analysis in minutes.

```bash
# 1. Clone the repository
git clone https://github.com/renpoo/TLU.git
cd TLU

# 2. Spin up the environment
docker compose up -d

# 3. Run the full pipeline with sample data
(bash bin/batch_unittest.sh : Unit Test)
bash bin/batch_generate_dummy_journal_data.sh
bash bin/batch_processing.sh
bash bin/batch_visualize_graphs.sh
```
