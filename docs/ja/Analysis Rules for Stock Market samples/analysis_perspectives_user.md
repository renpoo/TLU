# 株式取引の多次元分析: ユーザーの視点 (The User Perspective)

このドキュメントは、TLUの物理エンジン内において **ユーザー（トレーダー、機関投資家、アルゴリズム）** を主要な実体（エンティティ）として扱う際に適用される分析フレームワークの概要を説明します。この視点では、トレーダーがアクター（行為者）であり、株式は単に彼らが相互作用するための「乗り物（ビークル）」に過ぎません。

## 1. 運動学 (行動プロファイリング)
ユーザーの取引履歴を物理的な軌道（トラジェクトリ）としてマッピングすることで、彼らの真の性質と意図を分類することができます。
- **速度 (スピード):** 注文実行の頻度。
- **加速度 (攻撃性):** ユーザーがポジションを構築、または投売り（ダンプ）する急速さ。加速度の突然のスパイクは、アグレッシブな成行注文、FOMO（取り残されることへの恐怖）、またはパニック売りを示しています。
- **粘性 (摩擦: Friction):** ユーザーの「保有時間（ホールドタイム）」や忍耐力を表します。
  - **低い粘性:** 摩擦のない環境で動作する高頻度取引（HFT）アルゴリズムやボット。
  - **高い粘性:** 機関投資家のような「バイ・アンド・ホールド」投資家、または清算（損切り）できない「捕まった（Trapped）」個人トレーダー。

## 2. 情報幾何学とフォレンジック (異常検出)
ユーザーの取引パターンを「確率の多様体（Probability manifold）」として扱うことで、意図の逸脱を数学的に証明することができます。
- **KL ダイバージェンス (行動ドリフト):** ユーザーの現在の取引戦略が、歴史的なベースラインからどれほど乖離（ドリフト）したかを測定します。突然のダイバージェンス（発散）は、しばしば主要な戦略の転換、アカウントの乗っ取り、またはインサイダー情報に基づく行動に先行します。
- **Z-Score 特異点 (ミクロ・フォレンジック):** ユーザーの取引高や頻度が数学的に過去の分布を逸脱した「特定の瞬間」を特定し、異常な介入（例：「パンプ（Pump：価格の吊り上げ）」を実行する扇動者）の正確なタイムスタンプをピンポイントで示します。

## 3. ネットワーク・トポロジーと位相力学 (影響力と共謀)
資金フロー・ネットワーク内におけるユーザーの「重力（Gravity）」を計算することで、彼らの真の市場支配力を暴くことができます。
- **固有ベクトル中心性 (Eigenvector Centrality):** 「クジラ（Whales：大口投資家）」を特定します。中心性の高いユーザーは、必ずしも最も多くの取引を行っているわけではありませんが、「最も接続されたノード」と取引しているため、市場の流動性に対して絶大な重力（引力）を持ちます。
- **位相シフトと同期 (共謀: Collusion):** 協調的な攻撃（Coordinated attacks）を検出します。もし一見独立している複数のユーザーが、位相の遅れがゼロ（0.0 位相ドリフト）で完全に同期した取引パターンを示した場合、それは人工的な同期（Artificial synchronization）を証明しています。これは、協調的なチャットルームでのパンプ（吊り上げ）や、自動化されたスウォーム（群れ）アルゴリズムの典型的な特徴です。
