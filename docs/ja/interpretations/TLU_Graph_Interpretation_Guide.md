# TLU グラフ解釈ガイド (TLU Graph Interpretation Guides)

> **"数学的射影とビジネスの現実との間の、翻訳レイヤー。"**

TLU は組織の構造的な健康状態を把握するために、50種類以上の可視化（ビジュアライゼーション）を生成します。このディレクトリは公式の **「ロゼッタストーン」** として機能し、システムが生成するすべてのグラフに対する高度に構造化された視覚的なガイドを提供します。

## このガイドの使い方

これらのガイドは、深い数学的公式を説明するためのものではありません（それについては `docs/physics/` のマニュアルを参照してください）。代わりに、これらのガイドは `.png` ファイルを見て次のように問う **アナリストやエンドユーザー** のために設計されています：

1. **これを視覚的にどう読めばいいのか？**
2. **異常（アノマリー）はどのように見えるのか？**
3. **これは実際のビジネスにとって何を意味するのか？**

## 解釈ディレクトリ

認知的な過負荷（Cognitive Overload）を防ぐため、ガイドはコア分析（フェーズ2）パイプラインの論理カテゴリ (000 - 005) ごとに分割されています。

* **[財務諸表 (カテゴリ 000_0)](000_0_Financial_Statements.md)**: ベースライン。物理学に飛び込む前に、標準的な B/S と P/L をどう読むか。
* **[古典力学と剛性 (カテゴリ 000)](000_Classical_Mechanics_and_Stiffness.md)**: 速度、加速度の3D位相空間、および構造剛性マトリックスの読み方。
* **[熱力学とエントロピー (カテゴリ 001)](001_Thermodynamics_and_Entropy.md)**: T-S線図と自由エネルギースタックの読み方。
* **[情報幾何学とフォレンジック (カテゴリ 002)](002_InfoGeometry_and_Forensics.md)**: Z-Score トポロジーとネットワーク多様体の読み方。
* **[応用運動学 (カテゴリ 003)](003_Kinematics.md)**: FK（順運動学）および IK（逆運動学）ロボット工学シミュレーションの読み方。
* **[制御工学と安定性 (カテゴリ 004)](004_ControlTheory_and_Stability.md)**: スペクトル半径と LQR 収束の読み方。
* **[信号とノイズ (カテゴリ 005)](005_Signals_and_Noise.md)**: 位相ドリフト・ヒートマップと共振周波数の読み方。
