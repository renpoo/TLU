# 002. 情報幾何学とフォレンジック (Info Geometry and Forensics)

これは **不正検出と監査** にとって最も重要なフェーズです。データ分布の形状（幾何学）を分析し、保存則を厳密に適用することで、TLU は従来の帳尻合わせの監査手法が見逃してしまうような隠れた操作（Manipulation）を検出します。

---

### 1. マクロ・フォレンジック Z-Score (`002_2_1__macro_forensics_dashboard.png`)

![002_2_1__macro_forensics_dashboard](../../readme_plots/002_2_1__macro_forensics_dashboard.png)

* **📊 視覚的構造**: 時間経過に伴うネットワーク全体の「Z-Score」をプロットする折れ線グラフ。
* **📐 物理理論**: キルヒホッフの電流則（エネルギー保存の法則）。お金は無から生み出されたり、消滅したりすることはできません。Z-Score は、元帳全体にわたるこの法則の違反の深刻さを数学的に測定します。
* **🚨 異常検出 (Anomaly Detection)**:
  * Z-Score が **2.0** または **3.0** を超えるスパイク。
* **💼 ビジネスへの翻訳**: **壊れた元帳**。誰かが借方と貸方が一致しない仕訳を強制的に入力したか、対応する仕訳なしにシステムからお金が吸い上げられました。これは、深刻なデータ入力エラー、あるいは完全な横領の最も強力な数学的証拠です。

### 2. ミクロ Z-Score / KLドリフト ヒートマップ (`002_2_2_1__micro_KL_drift_heatmap.png`, `002_2_2_2__micro_Z_Score_heatmap.png`)

![002_2_2_1__micro_KL_drift_heatmap](../../readme_plots/002_2_2_1__micro_KL_drift_heatmap.png)

![002_2_2_2__micro_Z_Score_heatmap](../../readme_plots/002_2_2_2__micro_Z_Score_heatmap.png)

* **📊 視覚的構造**: マトリックス・ヒートマップ。X軸は時間、Y軸は特定の口座。濃い赤色 / 明るい色は高いストレスを示します。
* **📐 物理理論**: 情報幾何学（カルバック・ライブラー情報量 / KL Divergence）。特定の口座の確率分布が、その歴史的な振る舞いと比較して、どれほど予期せず「突然変異（Mutated）」したかを測定します。
* **🚨 異常検出 (Anomaly Detection)**:
  * 特定の口座（例: `ACC_Payroll_Exp（給与計算）`）が、特定の時間インデックス $t$ において突然明るい赤色に点灯する。
* **💼 ビジネスへの翻訳**: **不正行為者の特定**。マクロ・ダッシュボードは「犯罪が起きたこと」を教えてくれますが、ミクロ・ヒートマップは「*正確にどの口座*が操作されたか」を教えてくれます。もし給与計算（Payroll）が赤く点灯した場合、幽霊社員（Ghost employees）や承認されていないボーナスの支払いを探してください。

*注: TLU は時間経過とともにこれらの画像のシーケンスを生成します。パラパラ漫画（フリップブック）のようにめくって見る必要があるかもしれません。*

* **📊 視覚的構造**: 線（資金フロー）によって接続されたノード（口座）のウェブ。線の太さはフローのボリューム（量）を表します。
* **📐 物理理論**: グラフ理論。組織の財務の実際の配管（Plumbing）を視覚化します。
* **🚨 異常検出 (Anomaly Detection)**:
  * ノードが突然激しく位置を変える、または、通常は相互作用しない2つのノードの間に太い赤い線が現れる。
* **💼 ビジネスへの翻訳**: **犯行現場の可視化**。資金が流用されている流れを物理的に「見る」ことができます。もし太い線が突然「在庫（Inventory）」と未知の「仮勘定（Suspense Account）」を直接結びつけた場合、あなたは在庫盗難の視覚的表現を目の当たりにしていることになります。

### 4. 多様体の次元性 / 有効ランク (`002_1_3__manifold_dimensionality.png`)

![002_1_3__manifold_dimensionality](../../readme_plots/002_1_3__manifold_dimensionality.png)

* **📊 視覚的構造**: 時間経過に伴う単一の指標（有効ランク：Effective Rank）を示す折れ線グラフ。
* **📐 物理理論**: 特異値分解 (SVD)。ネットワークの真の「数学的次元数」を計算します。健康な組織が機能するためには、複数の次元（売上、給与、在庫など）が必要です。
* **🚨 異常検出 (Anomaly Detection)**:
  * 次元数のラインが突然急降下する（例: 8次元から一気に2次元に落ち込む）。
* **💼 ビジネスへの翻訳**: **人為的な中央集権化**。組織の自然な多様性が崩壊しました。すべての財務活動が単一のボトルネックに強制的に押し込まれています。これは **循環取引（Wash Trading）** の典型的な兆候です。偽の取引が単一の口座セットを通じて高速でループされ、人工的にボリュームを膨らませ、ビジネスの他の部分を飢えさせている状態です。
